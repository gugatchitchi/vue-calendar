# vue-calendar
Here we try to register one of the most popular plugins for vue - vietify.

We explore usage of vuetify through working with its calendar component.

Vuetify is great plugin to easily make material design website, however for complex design works it is not recomended. I have had a lot of trouble rewriting vuetify theme, which tooks qite a while. Hovewer component library is huge, which is great for leveraging front architecrute and concentrateon back-end.

### Compiles and hot-reloads for development

npm run serve
